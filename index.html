Como Formatar um computador

  • Faça um pendrive bootavel

    1. O Rufus é um dos melhores apps disponíveis a tarefa de criação de pen drives bootáveis. Além de ser portátil – funciona sem instalação –, o aplicativo pode alcançar o drive USB na busca de defeitos, como blocos irregulares e outras inconsistências que podem condenar o pen drive a médio prazo.
    2. Baixe o arquivo iso do windows aqui. Apos baixado abra o Rufus e selecione o arquivo e o pendrive (sertivique que o pendrive tenha pelo menos 8 gb).

    3. Assim que a instalação acabar feche o Rufus e sertivique que ocorreu tudo como esperado.

  • Configura a bios para dar boot com o pendrive

    1. Para entrar nas configurações da bios reenicie o computador e fique no momento que aparecer as iformações da placa mãe aperte a tecla responsavel pela finção, geralmente é 'F2' ou 'delete'.
    2. Assim que entre na bios, va para as opções de boot , e selecione o pendrive como boot primario. Salve e saia.
    3. Quando o computador ligar ja estará iniciando com o pedrive.

  • Instalar o windows ®

    1. Selecione o Windows 10 pro x64 ou Windows 10 pro x84, de acordo com seu hardware.
    2. Crie partiçoes no disco de acordo como for o seu uso (Dica: crie duas partições, uma para para o sistema e outra para colocar sues arquivos pessoais).
    3. Apos isso i windows sera instalado, quando terminar o computador reiniciara ja aparecerar para vc fazer os primeiros passos do Windows comfigure ele de acordo com suas preferencias. Pronto Computador Formatado!
